# 경쟁사 분석

## 주요 경쟁 서비스

### 1. Canny
- **가격**: $99/월~
- **타겟**: 성장하는 B2B SaaS 스타트업
- **설립**: 2017년

| 특장점 | 설명 |
|--------|------|
| Feature Voting | 사용자가 기능 요청에 투표 가능 |
| Public Roadmap | 공개 로드맵으로 투명성 확보 |
| Changelog | 업데이트 내역 자동 알림 |
| 연동 | Jira, Slack, GitHub, Intercom 등 |

### 2. UserVoice
- **가격**: $499/월~
- **타겟**: 대기업, 엔터프라이즈
- **설립**: 2008년

| 특장점 | 설명 |
|--------|------|
| 고급 세분화 | 사용자 세그먼트별 피드백 분석 |
| NPS/CSAT | 만족도 조사 내장 |
| 엔터프라이즈 기능 | SSO, 권한 관리, SLA |

### 3. Hotjar
- **가격**: 무료~€389/월
- **타겟**: UX/제품팀
- **설립**: 2014년

| 특장점 | 설명 |
|--------|------|
| Heatmaps | 클릭/스크롤 히트맵 |
| Session Recording | 사용자 세션 녹화 |
| Visual Feedback | 화면에 직접 피드백 표시 |
| Surveys | 인앱 설문조사 |

### 4. Gleap
- **가격**: $29/월~ (무료 플랜 있음)
- **타겟**: 웹 + 모바일 앱 개발팀
- **특징**: AI 기반

| 특장점 | 설명 |
|--------|------|
| AI 자동 응답 | 80% 질문 자동 해결 (Kai AI) |
| 60초 리플레이 | 버그 발생 전 60초 녹화 |
| 스크린샷 + 주석 | 사용자가 직접 마킹 |
| 올인원 | 버그리포트, 서베이, 로드맵 통합 |

## 가격 비교

| 서비스 | 무료 플랜 | 시작 가격 | 엔터프라이즈 |
|--------|----------|-----------|-------------|
| Canny | X | $99/월 | 문의 |
| UserVoice | X | $499/월 | 문의 |
| Hotjar | O | €32/월 | €389/월 |
| Gleap | O | $29/월 | 문의 |

## 시장 기회

1. **가격 격차**: Canny($99)와 Gleap($29) 사이에 포지셔닝 가능
2. **한국 시장 부재**: 모두 해외 서비스, 한국어/한국 결제 미지원
3. **복잡성**: 대부분 기능이 많아 설치/설정 복잡
4. **번들 크기**: Gleap 등은 위젯이 무거움

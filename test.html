우측 하단의 Feedback 버튼을 클릭하여 위젯을 테스트하세요.

Feature 1

이것은 테스트용 더미 컨텐츠입니다. 실제 서비스에서는 고객의 웹사이트가 됩니다.

Feature 2

피드백 위젯을 통해 사용자가 버그 리포트나 문의를 남길 수 있습니다.

Feature 3

제출된 피드백은 어드민 대시보드에서 확인할 수 있습니다.

테스트 방법

  1. pnpm dev로 서버 실행
  2. http://localhost:3000/signup에서 회원가입
  3. 조직 생성 후 프로젝트 생성
  4. 아래 스크립트의 data-project-id를 실제 ID로 변경
  5. npx serve .로 이 페이지 열기 (또는 Live Server 사용)
  6. 우측 하단 Feedback 버튼 클릭!
